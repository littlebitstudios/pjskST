# Project Sekai Score Tracker
This is a web app that calculates an unofficial technical score and rating for Project Sekai.

# Privacy
No data leaves your browser. For on-device persistence, the browser's LocalStorage is used. Import and export buttons allow you to use JSON files for transferring between devices.
